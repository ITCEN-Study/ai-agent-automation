# 📖 AI 협업형 IT 트렌드 & AX 풀스택 스터디 계획서

이 문서는 AI 에이전트(Antigravity)와 사용자(풀스택 개발자 지망생)가 협업하여 진행하는 **IT 기술 트렌드 및 AX 실무 역량 강화 스터디**의 운영 계획과 가이드라인을 정의합니다.

---

## 🤝 스터디 진행 프로세스

우리는 귀찮은 자료 조사와 문서 셋업은 AI에게 맡기고, 실제 설계와 코딩 실습은 개발자가 100% 손으로 직접 짜는 **효율성 극대화 방식**으로 진행합니다.

```text
┌──────────────────────┐      ┌──────────────────────┐      ┌──────────────────────┐
│  1. 아티클 선정 & 셋업 │ ───> │  2. 개념 학습 (Docs)  │ ───> │  3. 직접 코딩 (Works)  │
│    (AI가 폴더 및 명세서    │      │    (AI가 정리한 문서로    │      │    (사용자가 요구사항을   │
│     자동 생성 배달)    │      │     핵심 개념 이해)    │      │     보고 직접 코드 구현)    │
└──────────────────────┘      └──────────────────────┘      └──────────────────────┘
                                                                        │
                                                                        ▼
                                                            ┌──────────────────────┐
                                                            │  4. 리뷰 & 커밋 (Git) │
                                                            │    (AI에게 피드백 받고    │
                                                            │     최종 깃허브 업로드)    │
                                                            └──────────────────────┘
```

### 1단계: 학습 주제 및 아티클 선정
- 사용자가 직접 흥미로운 IT 아티클(예: 요즘IT 기사, 토스/당근 테크 블로그 등)을 제안하거나, AI에게 최신 트렌드 기사 추천을 요청합니다.
- AI는 기사를 읽고 분석하여 로컬 저장소에 자동으로 공부방을 셋업합니다.
  - `docs/topic-[주제명]/README.md` (기사 요약 및 학습 가이드)
  - `works/topic-[주제명]/requirements.md` (실습 구현 요구사항 정의서)

### 2단계: 개념 학습 (Docs)
- 개발자는 AI가 배달해 둔 `docs/`의 학습 노트를 읽고 핵심 기술 개념과 작동 원리를 이해합니다.
- 기사 원문을 직접 읽으며 추가 공부 내용을 적어 넣습니다.

### 3단계: 직접 코딩 및 실습 (Works)
- 개발자는 `works/` 폴더에 제안된 **요구사항 명세서**를 분석합니다.
- 뼈대 코드를 복사 붙여넣기 하지 않고, `npm init`부터 시작하여 디렉토리 설계, 서버 구축, API 연동, 화면 구현까지 **직접 밑바닥부터 자율적으로 구현**합니다.

### 4단계: 검증 및 코드 리뷰
- 코딩 실습이 완료되면, AI에게 구현된 코드를 보여주고 예외 처리나 성능 최적화 관점에서 피드백(코드 리뷰)을 받습니다.
- 피드백을 반영하여 코드를 개선하고 `git commit`하여 나만의 성장 기록을 차곡차곡 남깁니다.

## 🎯 학습 지향점

1. **실무형 문제 해결 능력 배양**: 단순히 튜토리얼을 따라 치는 것이 아니라, 정의된 요구사항 명세서를 보고 스스로 설계하고 문제를 해결하는 훈련을 합니다.
2. **트렌디한 기술 스택 탐색**: AI, AX(AI 전환), 마이크로서비스, 실시간 통신 등 현대 IT 프로덕트가 고민하는 최신 기술 도메인을 적극적으로 탐색합니다.
3. **탄탄한 깃허브 포트폴리오**: 매주 정리된 Docs와 직접 작성한 고품질 소스코드가 깃허브에 잔디와 함께 누적되어 취업 준비의 확실한 무기가 됩니다.
